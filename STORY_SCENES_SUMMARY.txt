├─────────────────────────────────────────────────────────────────────┤
│ 💬 "She should've met us already... Yeah... deserts are really      │

│ SCENE 2: DETERMINED LITTLE DRIVER                                   │
├─────────────────────────────────────────────────────────────────────┤
│ 💬 "I won't stop."                                                  │
│ 📚 Grammar: Future Negative (won't = will not)                      │
│ 🎯 Focus: Determination, future tense                               │
│ 🇹🇭 ฉันจะไม่หยุด                                                    │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ SCENE 3: FUZZY'S WORRY                                              │
├─────────────────────────────────────────────────────────────────────┤
│ 💬 "I don't like this part of the adventure..."                     │
│ 📚 Grammar: Present Negative (don't = do not)                       │
│ 🎯 Focus: Expressing feelings, present tense                        │
│ 🇹🇭 ฉันไม่ชอบส่วนนี้ของการผจญภัย...                                  │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ SCENE 4: ROUGH ROAD                                                 │
├─────────────────────────────────────────────────────────────────────┤
│ 💬 "AH! Desert pothole! Sorry! Hang on!"                            │
│ 📚 Grammar: Imperative + Phrasal Verb                               │
│ 🎯 Focus: Commands, phrasal verbs (hang on)                         │
│ 🇹🇭 อ๊ะ! หลุมในทะเลทราย! ขอโทษ! เกาะไว้!                            │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ SCENE 5: DOUBT CREEPS IN                                            │
├─────────────────────────────────────────────────────────────────────┤
│ 💬 "What if we can't find her...?"                                  │
│ 📚 Grammar: Hypothetical Question (what if + can't)                 │
│ 🎯 Focus: Expressing worry, modal verbs                             │
│ 🇹🇭 ถ้าเราหาเธอไม่เจอล่ะ...?                                        │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ SCENE 6: FUZZY COMFORTS HER                                         │
├─────────────────────────────────────────────────────────────────────┤
│ 💬 "We always find our friends. That's our thing."                  │
│ 📚 Grammar: Frequency Adverbs (always) + Possessive                 │
│ 🎯 Focus: Habitual actions, idioms                                  │
│ 🇹🇭 เรามักจะหาเพื่อนของเราเจอเสมอ นั่นคือสิ่งที่เราทำ                │
└─────────────────────────────────────────────────────────────────────┘

│ 💬 "Hold on! We're coming!"                                         │
│ 📚 Grammar: Imperative + Present Continuous                         │
│ 🎯 Focus: Phrasal verbs, continuous for future                      │
│ 🇹🇭 เกาะไว้! เรากำลังมา!                                            │
